# 領域移動ゲーム

WASDキーでプレイヤーを移動させ、A領域、B領域、C領域を移動できるゲームです。

## 必要な環境

- モダンなWebブラウザ（Chrome、Firefox、Edge、Safariなど）

## 実行方法

`index.html`をブラウザで開くだけです。

ダブルクリックするか、ブラウザにドラッグ&ドロップしてください。

## 操作方法

- **W**: 上に移動
- **A**: 左に移動
- **S**: 下に移動
- **D**: 右に移動

## ゲームの説明

- プレイヤー（赤い四角）はWASDキーで移動できます
- 3つの領域（A、B、C）があり、それぞれの領域の地面にはアルファベットが表示されています
- プレイヤーはこれらの領域内でのみ移動できます
- 画面下部に現在いる領域が表示されます
